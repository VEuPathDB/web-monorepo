# Frame 01 — Option A: three buttons in their own row

## Reference images

`base-user-comments.png` is uploaded in this chat — use it as the style reference for all frames in this session.

## Style guide

Match the visual style, fonts, colors, spacing, and UI component style shown in `base-user-comments.png` exactly. These screenshots are from a real production application — reproduce their look and feel faithfully.

Key style notes:

- PlasmoDB teal/blue header bar, white content area, clean sans-serif font throughout
- Section heading "User Comments" in dark gray, bold
- Small plain text links in teal ("Download", "Discuss", "Add a comment ✏️") — no button chrome, just styled anchor links
- Table with light gray striped rows, teal link text in cells, small gray header labels
- Tight vertical spacing in the section header area

## What to create

Show the same gene page User Comments section as in `base-user-comments.png`, with this change to the header area:

**Replace** the single "Add a comment ✏️" text link with a row of three small buttons sitting below the Download / Discuss links. The three buttons are styled consistently — use a compact outlined button style (light gray border, white background, dark gray text, small border radius). They sit on their own line, left-aligned, with a small gap between them:

1. **"Add a comment"** — leftmost button
2. **"Add AI-assisted comment"** — middle button, with a small "Beta" badge (a tiny teal rounded pill with white text "Beta") appended inline after the label
3. _(optional third action — leave as a placeholder button labelled "Discuss" if a third button makes the row look better balanced, otherwise just the two)_

Use realistic content in the table rows (copy from the base screenshot). Keep everything else on the page identical.

## Annotations

Figma-style: purple-filled circle with white number, thin connecting line, purple rounded-rectangle callout bubble.

- ① **Three-button row** — replaces the single "Add a comment" text link; consistent outlined button style
- ② **"Beta" badge** — small teal pill on the AI-assisted button signals the feature is new
