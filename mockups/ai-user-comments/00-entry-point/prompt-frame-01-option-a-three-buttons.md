# Frame 01 — Option A: two buttons in their own row (v2)

## Reference images

`base-user-comments.png` is uploaded in this chat — use it as the style reference for all frames in this session.

## Style guide

Match the visual style, fonts, colors, spacing, and UI component style shown in `base-user-comments.png` exactly.

Key style notes:

- PlasmoDB teal/blue header bar, white content area, clean sans-serif font throughout
- Section heading "User Comments" in dark gray, bold
- Small plain teal text links ("Download", "Discuss") — no button chrome at all
- Table with light gray striped rows, teal link text in cells

## What to create

Show the same gene page User Comments section as in `base-user-comments.png`. Make exactly the following changes to the header area — everything else stays identical:

**Row 1 (unchanged):** Keep the existing "⬇ Download" and "🗑 Discuss" as plain teal text links, exactly as they appear in the reference. Do not alter or remove either of these.

**Row 2 (new):** Below row 1, add a new line containing exactly **two** small outlined buttons, left-aligned, with ~8px gap between them:

- Button 1: labelled **"Add a comment"**
- Button 2: labelled **"Add AI-assisted comment"** with a small teal rounded pill badge reading **"Beta"** inline after the label text

Both buttons must be styled identically: white background, light gray border (1px solid), dark gray label text, small border radius (~4px), compact padding (~6px 12px). They must look like buttons — not text links.

**Remove** the original "Add a comment ✏️" text link that appears in the reference — it is replaced entirely by the two buttons above. Do not add any other buttons or links.

## Annotations

Figma-style: purple-filled circle with white number, thin connecting line, purple rounded-rectangle callout bubble.

- ① **Two-button row** — new row below the unchanged Download/Discuss links; both buttons have identical outlined style
- ② **"Beta" badge on button 2** — small teal rounded pill signals the AI feature is new
