1. Overview

We inherited the gene naming convention from the sequencing center which originally sequenced the genome and submitted to the GenBank. The following is an example: 

LOCUS       AAEE01000011  177983 bp   DNA  linear   INV 16-JUN-2006
DEFINITION  Cryptosporidium parvum chromosome 8 chr8.s1, 
            whole genome shotgun sequence.
ACCESSION   AAEE01000011 AAEE01000000
VERSION     AAEE01000011.1  GI:46226459
PROJECT     GenomeProject:144
KEYWORDS    WGS.
SOURCE      Cryptosporidium parvum
ORGANISM    Cryptosporidium parvum
            Eukaryota; Alveolata; Apicomplexa; Coccidia; Eimeriida;
            Cryptosporidiidae; Cryptosporidium.

              ......

  gene        <1686..>2558
              /locus_tag="cgd8_10"
  CDS         1686..2558
              /locus_tag="cgd8_10"
              /codon_start=1
              /product="uncharacterized protein with paralogs in
              cryptosporidium"
              /protein_id="EAK87454.1"
              /db_xref="GI:46226460"
              /translation="MSSTSRNPGPGGEIRPIPSFPMFDEERGNREFLVVTIEKEMEIS
              VPFLANYETAVDAGKFKRLIAYEHKADMYHIVKIENQTSDVQIFNENGKLKFVFSTHV
              FGINTEKILEKMSGYQDDLNKVVVEEFPFENIPNKPQYYNRGYKILIPENFDVFRDQQ
              YIEVTGQDGVEFKGIFSLDQPELKLPTEHTGDTYEFYTTSGTYAVFRGYENKRIYAIP
              VLEKVKCYVDTNYGRVRMANVCNLTFNVNYGILSVGEMEINTVNICGKRAIIGNTIEE
              VINIGISRLNEGSD"

The organism has been registered as 'cgd' at NCBI. Therefore, every gene starts with 'cgd' + chromosome number + '_' + sequential number. The sequential number is increased by 10 and is unique on a chromsome.

If there is a new gene identified between two adjacent genes, such as cgd8_350, cgd8_360. Therefore, the new seqential number will choose the median and the new gene name will be cgd8_355.
