Gene Model and Functional Curation
Standard Operational Procedures (SOP)

Overview

The following is completed documentation of many of ApiDB's standard operating procedures (SOPs) for gene model and functional curation. Each SOP outlines the purpose and scope of each process or procedure, all requirements necessary to carry out each procedure, a detailed description of the process or analysis, and all measures utilized to ensure that data and data types generated are consistent, current, and maintain specific quality requirements.
